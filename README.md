# album
一个简单的电子相册

### 功能：
* 点击触摸屏左右两边，浏览相册的上/下张图片
* 图片格式支持：jpg/jpeg


### 编程环境
* Ubuntu16.04
* [GEC6818开发板](https://item.taobao.com/item.htm?spm=a1z10.1-c.w5003-6427619857.1.45ee61bboSErWr&id=38029903389&scene=taobao_shop)
