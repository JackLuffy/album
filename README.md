# album
## 一个简单的电子相册

## 硬件平台：
* GEC-6818
* ARM-Cortex A53

## 软件平台：
* linux-3.4.39
* libc-2.23

## 功能：
* 点击触摸屏左右两边，浏览相册的上/下张图片
* 图片格式支持：jpg/jpeg
